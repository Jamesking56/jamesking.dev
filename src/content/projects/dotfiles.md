- **Neovim** - Editor setup with plugins and keybindings
- **Ghostty** - Terminal emulator config
- **Shell** - Bash/zsh configurations
- **Waybar** - Status bar for Hyprland
- **Misc** - Other Linux tools and utilities

Everything I need to set up my Omarchy Linux workspace from scratch.
